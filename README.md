###Step 3:

###Step 4:
End the program

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: Thrinesh Royal.T
#RegisterNumber:23004810
import numpy as np
a=np.array([[6,2,3],[3,1,1],[10,3,4]])
inverse=np.linalg.inv(a)
print(inverse)
```
